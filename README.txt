
Motion_analysis_KR README

Hello! Welcome to the accompanying R analysis code for the paper 'Common Phenomenal and Neural Substrate Geometry'. Here are some few tips. \
\
All the data is sorted in the data folder.
\
Ensure you have Outline enabled at the top right of the source panel in R studio, this way you can see the chunk labels and can jump around sections of the code. \
\
Second, ensure you load all the packages at the start (there are more to come later in each chunk, but be sure these base ones are right. \
\
Third, when it comes to load your data, ensure you check the file paths. \
\
Fourth, there are a number of hard-coded elements in the first few chunks that you will have to address. Please read through the code and comments carefully. Pay attention to how the data is handled, because we used a double-pass where the same stimulus conditions were repeated twice to check for correlation. This consisted of two blocks of 150 trials. \
\
Fifth, ensure you finish the first steps in the early chunks because they set up the data frames and some functions that are used later. \
\
Good luck! \
